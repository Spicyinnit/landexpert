# LandExpert - Full Working Next.js Site

Bu proje deploy-ready haldedir.

## Kurulum
1. ZIP'i aç
2. Klasörü GitHub repo'na yükle
3. Vercel'de import et
4. Deploy'a bas

## Local çalıştırma
```bash
npm install
npm run dev
```

## Yapı
- `app/` → Next.js app router
- `components/LandExpertWebsite.jsx` → ana site componenti
- `app/globals.css` → Tailwind giriş dosyası
